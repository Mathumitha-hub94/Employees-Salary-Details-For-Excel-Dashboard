# Employees-Salary-Details-For-Excel-Dashboard

## Objective ##
*This project aims to create an interactive Excel dashboard that analyzes salary data across different salary and regions.*

### Dataset Description: ###
*This dataset contains detailed information about employees joining an organization.
It consists of 151 records and 9 columns.*

**Key features**
  1. Remove Duplicate file
  2. Using "Substitute Function" to fill Name and Email-id columns
  3. Using Average value to fill Salary N/A values columns
  4. Descriptive Statistics
  5. Convert pivot table and pivot chart
  6. Using slicer
  7. Finally to create Salary dashboard.

 **conclution**
 
 *The analysis of the employee dataset provided meaningful insights into performance rating and employee satisfaction. Through descriptive analytics, we identified patterns related to employee distribution by department and salary levels.*
